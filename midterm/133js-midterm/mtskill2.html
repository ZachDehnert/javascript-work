  • Create a script that changes the background color of the body of this web page to various shades of purple as the user changes the range slider.

The slider is already created with a step of 6. Background colors can be set with CSS using the red RGB syntax of rgb(r,g,b). If g is set to 0, changing r and b will change shades of purple as long as r and b are equal values. r and b can each be 0 - 255.
