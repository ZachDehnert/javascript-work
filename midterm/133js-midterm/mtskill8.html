skill 8 (extra credit)

  • This final skill is extra credit.
  • Make edits to scripts you created in the other skills.
  • Pick three edits from the following list...
    • Instead of solid red or blue background colors (skill 1), the background colors use some type of gradient and the user is presented a third reset button to reset/remove the color choice.
    • Instead of strong purple colors (skill 2), the user can check a box to soften the shade. This softening can be done by adding some green to the color mix. However, the green value should be about half of the selected purple values. Example: rgb(24,12,24) and rgb(252,126,252).
    • Instead of just getting a random number (skill 3), the user is provided two textboxes where they can enter their own lower and upper limits. Those textbox values are used to produce the random number limitations.
    • Instead of removing the button (skill 4), the button tries to escape the user when they've clicked it more than 10 times. The button can move around the web page with each additional click.
    • Modify any of the money displays (skills 5 AND 6) to include commas as 1000 seperators. Do a little research online and find out how you can properly display a comma for every 1000 (example: 1,000; 10,000; 100,000; 1,000,000).
    • Reward the successful guesser (skill 8) with some fireworks. An animated gif of fireworks is included with your files. If the user guesses correctly within ten (10) tries, the fireworks display in some prominent location on the page.
  • For this page, declare variables and asign strings that describe the changes you made.
  • When the user clicks a button, your three changes are displayed in the output area. One sentence for each change is enough.

This skill is worth 12 points on top of your exam points from the other skills
