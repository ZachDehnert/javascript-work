skill 7

  • Create a number guessing game with difficulty levels.
  • The user can choose a level of difficulty. Based on the choice, the div boxes beneath the slider will light up as green, yellow, orange, or red (background color). Only the difficulty level color shows, while others are no background or light gray.
  • Based on the difficulty, a random number is generated from 1-10, 1-50, 1-100, or 1-1000 when the user clicks the 'start game' button. Note: the random number should display in the console.log() so that someone can cheat if they have the console displayed.
  • The game will keep track of the guesses, and display the guess count in the output area.
  • The game will let the user know if their guess is too high, or too low.
  • If the user cannot guess the number in ten (10) guesses, the game will...
    • reset the guess count to zero
    • Thank the user for playing
    • Automatically decrease the difficulty level by one level
    • Suggest the user play again
  • If the user guesses the number correctly within ten (10) guesses, the user is congratulated (text in the output area).
