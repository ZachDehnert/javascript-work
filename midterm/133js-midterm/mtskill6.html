skill 6

  • Create another script that demonstates the rule of 72 to the user.
  • As the user adjusts sliders, the initial investment and the rate of return will change. Note the steps and limits have already been applied to the sliders.
  • As the user adjusts sliders, a complete and dynamically changing sentence will display in the output area. The sentence will indicate the users selections AND the doubled amount AND when the doubling will occur. Example: If you invest $25000 at 7% return, you will have $50000 in 10.2 years.
  • Additionally, as the user decreases/incrases either slider, the money bag image (top right corner of web page) will shrink and grow.
  • For the money bag, adjust the width (the height will change proportionately) using some value determined from a math formula that involves both the investment amound and the return amount. If the user increases either slider, the total benefit will increase, and the money bag should get bigger.

Use toString(1) to limit the years decimal to 1 place.

This skill is worth 14 points
